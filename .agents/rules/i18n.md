# Scoped Rule: i18n

Activation scope:
- `app/[lang]/**/*.tsx`
- `app/[lang]/**/*.ts`
- `components/**/*.tsx`
- `components/**/*.ts`
- `dictionaries/*.json`
- `middleware.ts`

When active, enforce:
1. No hardcoded user-facing KR/EN copy in components and pages.
2. All visible strings must come from dictionary contracts (`dictionaries/ko.json`, `dictionaries/en.json`) or established text utilities.
3. Preserve locale routing and detection through `middleware.ts`.
4. Keep metadata/generateMetadata locale-aware for SEO.
5. Do not break `/ko` and `/en` parity when adding features.

Quality checks:
- Confirm both locales render equivalent structure and intent.
- Avoid introducing locale-dependent logic in UI without dictionary fallback.
