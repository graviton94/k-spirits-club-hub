# Graphify Knowledge Base

## Core Command
```bash
/graphify .
```

## Key Capabilities for K-Spirits
- **Architectural Mapping**: Finds "God Nodes" (central data models/logic) and "Surprising Connections".
- **Semantic Pathfinding**: `graphify path "Auth" "SpiritDetail"` -> Traces logic dependencies.
- **Always-on Rules**: `graphify antigravity install` -> Injects graph-awareness into my (Antigravity's) reasoning.
- **Wiki Generation**: `--wiki` -> Creates an agent-navigable documentation site.

## Navigation Strategies
- **GRAPH_REPORT.md**: Read this FIRST at the start of a session for high-level structure.
- **graph.json**: Use via MCP for deep-node queries during debugging.
- **Rationale Extraction**: Finds `# WHY:` comments and extracts them as rationale nodes.

## Team Flow
- Commit `graphify-out/GRAPH_REPORT.md` and `graph.json` so the agent always starts with the same mental model.
- Put `graphify-out/cache/` in `.gitignore` to keep repo light.
