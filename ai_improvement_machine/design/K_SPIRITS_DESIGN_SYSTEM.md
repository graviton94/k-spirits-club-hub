# K-Spirits Design System: "WOW" Aesthetic Bible

This document codifies the design principles for K-Spirits Club Hub to ensure a premium, state-of-the-art administrative and user interface.

## 🎨 Color Palette (The Deep Soul)
- **Primary (Night Shadow)**: `hsl(230, 40%, 10%)` - For deep, obsidian backgrounds.
- **Secondary (Soul Gold)**: `hsl(45, 80%, 45%)` - For elegant accents (Buttons, Rating stars).
- **Surface (Glassmorphic)**: `hsla(230, 25%, 20%, 0.6)` with `backdrop-filter: blur(12px)`.
- **Text (Moonlight)**: `hsl(215, 20%, 92%)` for readability on dark surfaces.

## ✍️ Typography (The Silent Voice)
- **Primary Typeface**: `Outfit` (Modern, geometric, premium).
- **Secondary Typeface**: `Inter` (Functional, clean for data grids).
- **Display Size**: `3.5rem` for section headers, with `-0.02em` letter-spacing.

## ✨ Micro-Animations (The Living Touch)
- **Hover Transitions**: `0.3s cubic-bezier(0.4, 0, 0.2, 1)`.
- **Entrance Effects**: Staggered fade-ins (20ms delay per list item).
- **Interactions**: Button clicks should have a slight `7%` scale-down effect for haptic feedback.

## 🖼️ Visual Hierarchy (The Clear Path)
- **Glassmorphism**: Use for modals and overlays to maintain spatial awareness.
- **Borders**: No solid borders. Use `1px solid hsla(0,0%,100%,0.1)` for "Invisible lines".
- **Gradients**: Subtle radial gradients to guide the eye towards "God Nodes".

## 🚀 WOW Implementation Rules
1. **No Pure Colors**: Never use `#000` or `#fff`. Use HSL for depth.
2. **Space as Luxury**: Minimum `2rem` padding for all containers.
3. **Intentional Blankness**: Each page must have one area of white/empty space to allow the eyes to rest.
